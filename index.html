Recent Projects

Travel Planner

With the Travel Planner App users can see all their travel details in one place. After logging in users can view the dashboard. There they can upload an avatar image and enter their travel date to get started. After entering a destination, a five day weather forecast, a map and the top 10 attractions for that destination will automatically be shown. Users can get more information about the sights by clicking on them. A to-do list is included to prepare for the upcoming trip. This app uses the locationIQ API, Darksky API, Mapbox API and Sygic Travel API. It is responsive across all screen sizes.

React (incl. React Router) / Node / Express / MongoDB / JWT / SCSS

Website Github

Momentum Clone

This productivity app is a clone of the famous Momentum Chrome Extension. It was created during a Chingu Voyage in collaboration with 2 other developers. Each time the app is opened, a beautiful new image is displayed by using the Unsplash API and users are greeted with one inspirational quote per day. The app displays the time and weather at the current location as well as a 5 day forecast. Users can keep track of tasks with to-do lists, save important links and search the web by choosing between Google, DuckDuckGo & Bing.

HTML / CSS / JavaScript / jQuery / Node / Express / EJS

Website Github

Book Finder

The Book Finder App lets users search for books using the Google Books API. An infinite scroll keeps pulling more search results. After creating a user account favorite books can be saved to a list. The app is responsive across screen sizes.

React (incl. React Redux & React Router) / Node / Express / MongoDB / JWT / CSS

Website Github

Task App

The Task App is a simple to-do list app. It has a clean and modern design. Create a user account and start keeping track of your tasks. Cross your tasks off the list when you're done or delete them. See all your tasks, only completed or only uncompleted ones. The app is responsive across screen sizes.

Node / Express / MongoDB / EJS / JWT / jQuery / HTML / CSS
